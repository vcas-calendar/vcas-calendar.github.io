バーチャルキャストイベントカレンダーについて

有志によって製作された、バーチャルキャストに関する予定を公開するための共有カレンダー (公認) です。

ライブ配信に限らず、集会や企画の応募締め切りなど、なんでも自由に登録できます。

権限などは必要なく、だれでも予定の登録は可能なので、ぜひご活用ください。

お問い合わせ

意見や要望などがあれば、 バーチャルキャスト非公式Discordの「#カレンダーについて」チャンネル へどうぞ。 カレンダーの編集権限については、ピン留めされたメッセージをご参照ください。
